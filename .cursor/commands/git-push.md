# git-push: Create branch, commit and push

Запусти скрипт `.cursor/scripts/git-push.sh` в терминале проекта.

Скрипт делает следующее:
1. Показывает `git status` — список изменённых файлов
2. Предлагает название ветки (на основе изменённых файлов + дата) — можно принять или ввести своё
3. Создаёт новую ветку и переключается на неё
4. Делает `git add -A` — добавляет все изменения
5. Предлагает короткое сообщение коммита (на основе затронутых модулей) — можно принять или ввести своё
6. Делает коммит и пуш в origin

**Запусти в терминале:**

```bash
bash .cursor/scripts/git-push.sh
```
